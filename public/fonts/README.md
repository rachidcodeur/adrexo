# Fichiers de police Mont

Placez les fichiers de police Mont dans ce dossier avec les noms suivants :

- `Mont-Regular.woff2` (weight: 400)
- `Mont-SemiBold.woff2` (weight: 600)
- `Mont-Bold.woff2` (weight: 700)

## Où télécharger Mont ?

Vous pouvez télécharger la police Mont depuis :
- Le site officiel du créateur
- Des sources de polices gratuites comme Font Squirrel, Google Fonts (si disponible), ou d'autres sites de polices

## Formats supportés

Le projet est configuré pour utiliser les fichiers `.woff2` qui offrent la meilleure compression et compatibilité.

Si vous avez d'autres formats (`.woff`, `.ttf`, `.otf`), vous pouvez les convertir en `.woff2` ou modifier la configuration dans `app/layout.tsx` pour utiliser ces formats.
